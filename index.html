Outfit Ready is an android and iOS app that allows users to see their closet/wardrobe in virtual form.

  • Users can simply take pictures and upload their clothing items to a virtual closet where they can create new clothing outfits.
  • Used Travis CI and a test driven development philosophy for unit testing.

Dheliver

Dheliver is a food delivery website for students at UCSC.

  • Used Stripe API for an easy checkout, Beautiful Soup for web scraping menu data from the university website, HTMLto create a clean UI.
